# 桃園龍潭大池生態與文化網｜真實照片田野版

## 使用方式
1. 用 VS Code 開啟整個 `longtan_lake_site` 資料夾。
2. 安裝 Live Server 外掛。
3. 打開 `index.html`，右鍵選 `Open with Live Server`。

## 重點
- 本版使用真實照片，不再是簡圖。
- 物種照片為代表照，請以現地觀察確認是否出現在龍潭大池。
- 水利編號與進出水口精確位置若公開資料無法確認，網站內已標示「待查」，請實地拍照或詢問管理單位。
- `sugarcube/longtan_task.twee` 可匯入 Twine，故事格式選 SugarCube 2。
