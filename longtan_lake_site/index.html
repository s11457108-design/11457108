建議繳交方式

用 VS Code 開啟整個資料夾，安裝 Live Server，從 index.html 啟動。田野照片若是你自己拍的，可直接覆蓋 assets/img 中的同名圖片，網站版面會自動套用。

若老師要求「本人實地拍攝」，請把本網站內的代表照片換成你自己的現場照片；目前版本使用的是可授權公開真實照片。
